### Patient-Healthcare
# 📌 Project Overview
This project focuses on analyzing patient healthcare data to monitor hospital performance, patient trends, and treatment outcomes. The dataset was cleaned, processed, and visualized to generate meaningful insights that help improve operational efficiency and patient care quality.
# 🎯 Project Objectives
Analyze patient admission data to identify trends and patterns.
Monitor key healthcare KPIs such as total patients, treatment cost, and average length of stay.
Identify the most common diseases and treatments.
Perform department-wise performance analysis.
Analyze patient demographics (age and gender distribution).
Detect abnormal lab results and critical health cases.
🛠️ Tools & Technologies Used
Microsoft Excel (Data Cleaning & Preparation)
SQL (Data Querying & KPI Calculation)
Power BI / Tableau (Dashboard & Visualization)
📊 Key KPIs
👩‍⚕️ Total Patients
🏥 Total Admissions
💰 Total Treatment Cost
⏳ Average Length of Stay
🧪 Abnormal Lab Results Count
📈 Department-wise Patient Distribution
# 📊 Dashboard Insights
Patient admission trends over time
Disease-wise and department-wise analysis
Gender and age group distribution
# 🚀 Conclusion
This project demonstrates how healthcare data analytics can improve hospital decision-making, optimize resource allocation, and enhance patient care through data-driven insights.
